iSpindel: DIY elektronische Bierspindel
  • aktuelle Gärung, mit letztem Messwert
  • aktuelle Gärung mit Datentabelle
  • einige Parameter mittels config_frontend.php gesetzt
  • Gärung Mai, eine Grafik, mehrere y-Achsen
  • Gärung Mai, eine Grafik je Variable
  • gesamter Zeitraum
  • gesetzte ResetFlags über gesamten Zeitraum
  • Gärung Mai, keine Grafiken, nur komplette Datentabelle
  • Gärung Mai, Winkel und Temperatur in einer Grafik
  • lchart.php Gärung Mai, Winkel und Temperatur in einer Grafik
  • plato4.php für iSpindel Version 4
  • Status der Batterie, Temperatur und Winkel als Messuhren
  • ResetFlag setzen

Parameter:

  • name: Name der iSpindel
  • varlist (bei dashboard.php): Kommagetrennte Liste der anzuzeigenden Variablen, z.B. Temperature, Battery und Gravity (erst ab Version 5 der iSpindel), Angle und ResetFlag
  • var1 und var2 (bei lchart,php): Erste und zweite anzuzeigende Variablen
  • box (bei dashboard.php): [0 oder 1], Boxen mit dem zuletzt gemessenen Wert werden angezeigt
  • maxis (bei dashboard.php): [0 oder 1], 0: mehrere Grafiken mit je einer Variablen; 1: eine Grafik mit mehreren y-Achsen
  • hours: Es werden die letzten x Stunden angezeigt
  • reset: [0 oder 1]: Es werden die Werte seit dem letzten ResetFlag angezeigt
  • date: (TT.MM.YYYY): Es werden die Werte zwischen 2 ResetFlags angezeigt
